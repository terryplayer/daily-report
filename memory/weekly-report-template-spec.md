# 📋 周复盘报告模板规范

> 最后确认：2026-05-24 周日 | 定版人：十三哥

## ⏰ 触发时间
- **每周日 12:00** 准时生成并发送
- 不允许提前，不允许延迟

## 📐 报告格式

### 整体风格
- 深色主题（Dark Mode），仿 GitHub Dark 风格
- 使用 `.section`, `.reco-card`, `.rating-card`, `.grid-2/3/4` 等布局组件
- 颜色体系：
  - 🔴 红色 = 上涨/买入/短期持有
  - 🟢 绿色 = 下跌/卖出/回避
  - 🟡 黄色 = 中性/关注
  - 🔵 蓝色 = 长期持有/板块标配
  - 🟣 紫色 = 板块标题

### 必须包含的8大板块

#### 板块一：本周大盘回顾
- 四大指数（上证/深证/创业板/科创50）周变化
- 市场总评及下周展望

#### 板块二：本周板块轮动回顾
- 领涨/领跌板块TOP及核心标的
- 驱动因素及轮动节奏分析

#### 板块三：预测验证 & 准确率评估
- 准确率评分卡（趋势方向/板块判断/个股判断/风险评估）
- RS排名TOP5/BOTTOM5（带RS值、周涨幅、评级、信号）
- 每日预测 vs 实际走势表格（含结果判定✅⚠️❌）
- 偏差原因分析

#### 板块四：本周信息汇总
- **宏观数据仪表盘** — PMI / CPI / PPI / GDP 卡片式展示
- **行业轮动周期阶段** — 美林时钟判断（复苏/过热/滞胀/衰退）+ 配置建议
- 国内政策/经济 — 财政、货币、监管等
- 国际市场 — 美股、美债、汇率、大宗商品
- 行业/公司动态
- **技术面信号** — 指数级技术指标（RSI/MACD/布林带/支撑压力/量能分析）
- 个股技术面量化说明（如果数据不足需要注明）

#### 板块五：下周趋势判断
- 各指数支撑/压力位表格
- 板块预判（看好/中性/回避）
- 风险提示（高风险/中等风险分层）

#### 板块六：下周板块及个股建议
- 重点推荐板块（带推荐逻辑）
- 持仓操作建议（26只全覆盖，按板块逐一给出）
- 下周操作计划（加仓/持有/减仓）
- 关键观察点

#### 板块七：建议新增持有个股
- 基于"中性及以上"板块，推荐**不在持仓中**的5只股票
- 每只包含：
  - 股票名称 + 代码
  - **推荐价位**（建仓区间 + 加仓区间，用 tag 标注）
  - **触发信号**（放量突破信号 🔴 + 回调企稳信号 🟢）
  - 推荐理由（引用行业趋势、政策利好、近期走势等）
  - 风险提示
- 综合配置建议 + 优先级排序

#### 板块八：综合评级与操作建议
- 🏆 综合评级说明（RS 20% + 多因子 30% + 波动 15% + 技术面 15% + 行业轮动 20%）

### 三档评级全覆盖26只持仓：
  - 🔥 建议短期持有（趋势交易/事件驱动）
  - 💎 建议长期持有（价值投资/行业龙头）
  - 🚫 建议卖出/减仓（趋势走坏/基本面恶化）
- 每个标的引用RS + 多因子数据作为依据
- 底部统计卡片：🔥 X只 | 💎 X只 | 🚫 X只 | 观望 X只

### 数据来源
- stock_analysis.py 的 RS排名、波动预警、多因子评分、技术面信号
- macro_fetch.py 的宏观数据
- 网页搜索获取最新的周末新闻
- 每日收盘简报中的历史数据

### 发送方式
- 先输出文字摘要（含评级统计）
- 再通过 send-feishu-file.sh 发送HTML附件
- 文件名格式：`A股周复盘_2026年第X周_完整版.html`
