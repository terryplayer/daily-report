# 📊 复盘汇总看板

## 📂 目录结构

| 目录 | 用途 |
|:----|:-----|
| `01-笔记/` | 学习笔记、日常记录 |
| `02-研究/` | 混动系统架构、工作流、研究笔记 |
| `03-持仓/` | 持仓分析 |
| `04-学习/` | 考试、中学生学习系统 |
| `05-工具/` | Python 小工具 |
| `_templates/` | Templater 模板（按 Alt+T 使用） |
| `memory/` | 每日笔记（小十三读写） |

## 📈 最近复盘

```dataview
TABLE date as "日期", tags as "标签"
FROM "02-研究" OR "memory"
WHERE contains(tags, "复盘") OR contains(file.name, "复盘")
SORT file.name DESC
LIMIT 10
```

- 按 `Alt+T` → 选「每日复盘」→ 生成当天复盘笔记
- 在 `memory/` 目录写日报
- 修改 `_templates/每日复盘.md` 调整模板
