# 🏗 知识库结构

> 这个 Obsidian 库 = 你的工作区（workspace）

```
workspace/                  ← Obsidian 库根目录（直接打开这个文件夹）
├── .obsidian/              ← Obsidian 配置文件（不用动）
├── _moc/                   ← 导航页（Map of Content）
│   ├── 总览.md             ← 首页入口 ⭐
│   ├── 日报索引.md         ← 日报导航
│   ├── 持仓分析.md         ← 持仓导航
│   └── 研究笔记索引.md     ← 研究笔记导航
├── _templates/             ← 模板
│   ├── 日记模板.md         ← 每日笔记
│   └── 日报模板.md         ← 日报结构
├── memory/                 ← 日记（已有）
│   ├── YYYY-MM-DD.md       ← 每日日记
│   ├── stock-watchlist.md  ← 持仓股票清单
│   └── ted-recommendations.md
├── 日报/                   ← 每日股票报告
│   └── YYYY-MM-DD/
│       ├── premarket.md    ← 盘前简报
│       └── daily-combined.md ← 收盘简报
├── 持仓/                   ← 个股分析笔记
├── 研究笔记/               ← 深度研究
└── daily-report-html/      ← HTML 版本（已有，供网页用）
```

## 核心工作流

1. **每天** → 日记自动生成（`memory/YYYY-MM-DD.md`）
2. **盘前/收盘** → 报告自动同步到 `日报/`
3. **需要时** → 查 `_moc/总览` → 跳转任何地方
4. **研究** → 在 `研究笔记/` 里写，通过 `[[链接]]` 关联

## 双向链接流

```
日记 ←→ 日报 ←→ 持仓 ←→ 研究笔记
  ↓                    ↑
  └──── 想法/总结 ─────┘
```
