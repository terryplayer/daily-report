## 核心脚本一览

| 脚本 | 用途 | 生成物 |
|:-----|:------|:-------|
| `scripts/stock_analysis.py` | 数据采集+模型计算 | `stock_analysis_cache.json` |
| `scripts/gen_premarket_report.py` | 标准盘前简报 | `premarket-YYYY-MM-DD.html` |
| `scripts/gen_midday_report.py` | 标准午间监测 | `midday-YYYY-MM-DD.html` |
| `scripts/gen_closing_report.py` | 标准收盘简报 | `daily-combined-YYYY-MM-DD.html` |
| `scripts/gen_weekly_report.py` | 标准周复盘 | `weekly-review-YYYY-MM-DD.html` |
| `scripts/gen_premarket_advanced.py` | ★ 盘前进阶版 | `premarket-advanced-YYYYMMDD.html` |
| `scripts/gen_advanced_reports.py` | ★ 午间/收盘/周复进阶 | `midday/closing/weekly-advanced-YYYYMMDD.html` |
| `scripts/gen_daily_snapshot.py` | ★ 每日行情快照(RAG) | FAISS向量索引 |
| `scripts/gen_dashboard.py` | ★ 报告中心dashboard | `dashboard.html` |
| `scripts/daily_score_gen.py` | 每日选股评分 | 选股评分数据 |
| `scripts/gen_index.py` | 主页index.html | `index.html` |
| `scripts/hermes_review.py` | Hermes预测复盘 | Hermes记忆 |
| `scripts/gh-pages-sync.sh` | Git提交→CF自动部署 | GitHub main → CF Pages |
| `scripts/obsidian-sync.sh` | Obsidian同步 | Obsidian仓库 |

## 部署流程

```
gen_*.py 生成HTML
        ↓
git add daily-report-html/*.html
git commit
git push github main
        ↓
Cloudflare Pages 自动部署（约30秒）
        ↓
https://daily-report-3ai.pages.dev/xxx.html
```

- **标准版** 在 📰🌤📊📮📈 五个时间点各部署一次
- **进阶版** 跟随标准版部署，不额外增加部署次数
- **dashboard** 在每日16:00和18:00各刷新一次，确保进阶版入口显示
