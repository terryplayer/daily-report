## 文件①：`calc_sector_leaders.py`

**输入：** `data/raw/YYYY-MM-DD.json`

**处理逻辑：**
1. 按 `f100` (细分行业) 分组
2. 每个行业内按 `f21` (流通市值) 降序排列
3. 取 TOP9
   - rank 1-3 → "🏆龙头"
   - rank 4-6 → "💎核心"
   - rank 7-9 → "📌关注"
4. 跳过流通市值 < 5亿

**输出：** `data/sector_leaders.json`
```json
{
  "date": "2026-06-10",
  "total_sectors": 111,
  "total_stocks": 999,
  "leaders": {
    "688549": {"rank": 1, "label": "🏆龙头", "tier": 1, "sector": "半导体", "name": "中巨芯-U", "circ_mv": 185.2}
  }
}
```
