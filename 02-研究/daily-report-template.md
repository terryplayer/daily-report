## 四、通用排版规范（所有飞书推送）

### 4.1 核心规则

| # | 规则 | 说明 |
|:-:|:-----|:------|
| 1 | 分隔线 | 板块之间用 20 个全角横线 `━━━━━━━━━━━━━━━━━━` |
| 2 | 空行 | 每个板块标题前后各空 1 行 |
| 3 | 对齐 | 表格列用 2 个空格间隔，左对齐 |
| 4 | 颜色 emoji | 🔴=涨/利好 🟢=跌/利空 |
| 5 | 长度限制 | 总内容 ≤ 2000 字符 |
| 6 | 持仓明细 | 仅展示 TOP5 涨 + TOP5 跌 |
| 7 | 末尾 | 小十三签名 + 在线链接 |
| 8 | 🚫 禁止 | `&amp;` 等 HTML 转义字符、内容重复、数据挤成一团 |

### 4.2 参考基准
> 📌 **以 2026-06-02（周二）的收盘简报为基准格式样板**
> 每次生成前先查看前一天的正确版本作为参考。

### 4.3 常见错误（❌ 禁止）

```
❌ 错误：板块数据挤在一起
💻 科技/半导体+1.9%21A↑TOP超配

✅ 正确：用空格对齐
💻 科技/半导体 +1.9%  21只  A  ↑TOP  ⭐超配

❌ 错误：HTML 转义字符
&amp;  → 应该用 &

❌ 错误：内容重复
同一段话出现两次
```

### 4.4 生成检查清单
生成完后逐条检查：
- [ ] 板块评分表格对齐了吗？（用空格隔开各列）
- [ ] 有 `&amp;` 吗？（应该替换为 `&`）
- [ ] 内容有重复吗？
- [ ] 分隔线用对了吗？
- [ ] 持仓明细只显示了 TOP5+TOP5？
- [ ] 末尾有签名和链接？
- [ ] 总长度在 2000 字以内？
