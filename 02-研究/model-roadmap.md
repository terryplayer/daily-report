## 一、统一模型度量框架（已完成框架，持续完善）

所有模型输出标准化格式，盘前/午间/收盘/周复盘共用：

```text
                  统一评估框架 (get_model_summary)
                  ┌──────────────────────────────┐
   RS排名 ──────→ │                              │
   MOM动量 ──────→ │  综合评分(0-100) + 信号+建议  │ ──→ 盘前简报
   MRD偏离 ──────→ │                              │ ──→ 午间监测
   BBW布林 ──────→ │  板块: {评分, 信号, 配置}      │ ──→ 收盘简报
   多因子 ──────→ │  个股: {评分, 信号, 操作}      │ ──→ 周复盘
   行业轮动 ─────→ │                              │
   随机森林 ─────→ │                              │
                   └──────────────────────────────┘
```

**权重分配（统一）**：RS 25% + 动量 20% + MRD 10% + 多因子 30% + 轮动 15%

**输出格式**：
| 评分 | 信号 | 配置建议 | 操作 |
|:---:|:----:|:--------|:----|
| ≥80 | 🔴🔴 | ⭐超配 | 加仓/持有 |
| 60-79 | 🔴 | ↑增配 | 关注/增持 |
| 40-59 | ⚪ | —标配 | 观望/持有 |
| 20-39 | 🟢 | ↓减配 | 减仓/回避 |
| <20 | 🟢🟢 | 末低配 | 止损/清仓 |
