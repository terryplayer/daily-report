## 第四阶段：随机森林分类器（数据积累后）

### 条件
- 需要 ≥60个交易日数据（当前11天）
- 预计 **2026年8月中旬** 数据够用
- 需要安装 `scikit-learn`（`pip install sklearn`）

### 实现方案

```python
# 特征矩阵（每只股票每天一行）
因子 = [
    RS_rank,       # RS相对强度排名（1-77）
    MOM_5,         # 5日动量
    MOM_10,        # 10日动量
    BBW,           # 布林带宽度
    MRD,           # 均值回归偏离度
    RSI,           # 相对强弱指标
    volume_ratio,  # 量比
    north_flow,    # 北向资金方向（+/-）
    overnight,     # 隔夜外盘涨跌幅
    PCR,           # 期权认购认沽比
]
# 标签: 次日涨/跌（二分类）
# 模型: RandomForestClassifier(n_estimators=200, max_depth=6)
```

### 预期效果
| 模型 | 单因子准确率 | 多因子综合 |
|:----|:-----------:|:---------:|
| RS排名 | ~52% | — |
| MOM | ~53% | — |
| MRD | ~55% | — |
| **随机森林（全部因子）** | — | **~62-68%** |

## 第五阶段：协整配对（数据积累 + 策略层）

### 条件
- 需要 ≥60个交易日
- 同板块有强相关性的股票对

### 原理
```text
找到两只同板块、长期价差稳定的股票：
当价差偏离均值2个标准差时 → 做多低估 + 做空高估
当价差回归均值时 → 平仓获利
```

### 候选配对（同板块）
| 板块 | 候选对 |
|:----|:-------|
| 通信/电子 | 天孚通信 ↔ 新易盛（光模块双雄）|
| 科技/半导体 | 长电科技 ↔ 华特气体（封测+材料）|
| 能源/公用 | 华电辽能 ↔ 中国西电（电力+设备）|

### 预期效果
- 不是方向预测，而是**对冲策略**
- 年化收益目标：8-15%（低风险套利）
