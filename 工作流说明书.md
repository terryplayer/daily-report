### ⏰ 9:00 — 盘前数据采集 + 简报预生成 ← 重活

| 项目 | 内容 |
|------|------|
| **cron 表达式** | `0 9 * * *` (每日) |
| **运行模式** | isolated, delivery=mode:none (静默不推送) |
| **超时** | 600秒 |

**执行步骤：**
1. 判断是否周末/节假日 → 周末则走【周末简报】流程
2. **工作日流程：**
   a. `stock_analysis.py --update` → 更新历史行情
   b. `tushare_fetch.py report` → 获取Tushare数据
   c. `stock_analysis.py --json` → 获取RS排名/多因子评分/波动预警/技术信号，缓存到 `/tmp/stock_analysis_cache.json`
   d. 若脚本失败 → `web_search` 兜底抓取美股/外围行情
   e. 基于数据生成**完整盘前简报文字内容**（8大板块）
   f. 生成HTML文件 → `daily-report-html/premarket-YYYY-MM-DD.html`
   g. 运行 `gh-pages-sync.sh` → 推送到 GitHub → Cloudflare Pages 自动部署
   h. 将文字内容保存到 `/tmp/premarket-content.txt`
3. **周末流程：**
   a. web_search 搜索周末资讯
   b. 生成周末市场简报文字（海外回顾+周末资讯+板块回顾+下周展望）
   c. 生成HTML → `daily-report-html/weekend-YYYY-MM-DD.html`
   d. 运行同步
   e. 保存内容到 `/tmp/premarket-content.txt`
4. 静默退出，不推送飞书消息

## 三、个股评级体系

### 5级盘前评级（RS+多因子+技术面综合）

| 评级 | 含义 | 条件 |
|------|------|------|
| 🔴🔴 强势持有 | 高置信度 | RS≥A + 多因子TOP5 + 板块评分领先 |
| 🔴 偏强持有 | 一般置信度 | 指标支持但未三重确认 |
| ⚪ 中性观察 | 无方向 | 无明确信号 |
| 🟢 偏弱关注 | 偏弱 | 技术面/资金面偏弱 |
| 🟢🟢 弱势减仓 | 明确下行 | 下行信号确认 |

### 评级修正规则
1. **板块联动**：个股强势但所属板块评分在后50%，下调一档
2. **量能验证**：放量上涨+强势→加置信度；缩量上涨→下调一档
3. **趋势vs事件**：趋势性→标注「趋势」；事件性→标注「事件驱动」
4. **数据不足**：默认按「中性偏谨慎」处理

### 周复盘综合评级（20%RS+30%多因子+15%波动+15%技术面+20%行业轮动）

| 类型 | 说明 |
|------|------|
| 🔥 建议短期持有 | 趋势交易/事件驱动/超跌反弹 |
| 💎 建议长期持有 | 价值投资/行业龙头/政策受益 |
| 🚫 建议卖出/减仓 | 趋势走坏/基本面恶化/波动过大 |
